## Com compilar i executar el projecte

1. clona aquest repo: `git clone https://github.com/jahosp/cov-response`
2. `cd cov-response`

### App
1. `cd app`
2. `npm install`
3. executa `ionic serve` a la terminal

### Middleware
1. `cd middleware`
2. `npm install`
3. executa `node index.js` a la terminal
